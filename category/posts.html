1. Neural Network, a code perspective¶

For a solution using neural network, there are three essentials:

  1. model definition [prototxt]
    It defines what the nerual network looks like, how many layers it contains.
  2. pretrained model [caffemodel]
    It can be regarded as initialized parameter value, including weights and bias. But it may not be trained specially for our own data. So we need to recompute these values based on our own dataset. And the output of training is also caffemodel file.
  3. data [imdb]
    It contains all the data information.

2. Data in faster_rcnn¶

In faster_rcnn code, Girshick use a basic class imdb to represent the image dataset. And for different dataset, use derived class, such as pascal_voc and coco. These files are stored in py-faster-rcnn/lib/datasets/. Let's have a look at how imdb work.

The most important member in imdb is roidb. roidb contains bounding box location, image path and related useful information. So what is roidb?

roidb is a list of dictionaries, each with some keys.

There are 10 key-value pairs for each image. They are divided into two groups: original and additional. Let's explain their meanings.

4.1 original¶

  1. boxes
    n*4, 2D array containing all the object boxes in the order of (x1, y1, x2, y2)
  2. gt_classes
    n, 1D array, containing the label of each box.
  3. gt_overlaps
    n*num_class, the relation between i-th box and j-th class label. "1" mean i-th box belongs to j-th class, otherwise 0. So it is a spase matrix. Use scipy.sparse.csr_matrix to store such sparse matrix.
  4. flipped
    True of False. Whether to use flipped image or not depend on config. Default is to use. So we have only 643 images, but the length of roidb is 1286.
  5. seg_areas
    n, 1D array, containing area of each box.

4.2 additional¶

  1. image
    image absolute path
  2. width
  3. height
  4. max_classes
    n, 1D array, containing the ground truth label(0,1,2,3,etc) of each box, since the ground truth label of the correspoinding box has maximum value.
  5. max_overlaps
    n, 1D array, containing the max probability of each box.

Original attributes are defined in py-faster-rcnn/lib/datasets/[derived_class].py, additional attributes are in py-faster-rcnn/lib/roi_data_layer/roidb.py---prepare_roidb() function.

5. Load your own data¶

5.0 outline¶

  • Create soft link to your dataset in py-faster-rcnn/data/.
  • Edit factory.py in py-faster-rcnn/lib/datasets/.
  • Create derived class file in py-faster-rcnn/lib/datasets/, such as pedestrian.py.
  • Test.

5.1 Step 1¶

duino@du:~$ ln -s [your dataset path]/insulator lib/datasets/insulator

5.2 Step 2¶

Add below to factory.py.

for year in ['2016']:
    for split in ['train', 'test']:
        name = 'insulator_{}_{}'.format(year, split)
        __sets[name] = (lambda split=split, year=year: insulator(split, year))
